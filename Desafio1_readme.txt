Para este desafio01, foi feito uma função que ler o ano e retorna o século em que 
aquele determinado ano se encontra.

Para o return, foi usado o Math.floor para retornar o menor número inteiro e também 
foi usado o %(retorna o resto que sobra), assim como também os :(operadores condicionais ternários).

O index.html foi usado somente para abrir o browser para poder testar no console, 
já que neste programa não foi utilizado o terminal.
